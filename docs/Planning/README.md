# Planning 文档目录

本目录用于存放产品愿景、架构取舍、运营路线、未来计划和讨论结论。

这里的文档不等同于立即执行的开发任务。它们用于帮助后续开发者理解：

- 为什么当前阶段做某个取舍。
- 哪些能力是未来必须补上的。
- 哪些事情现在可以先不做。
- 哪些设计会影响长期产品形态。
- 哪些系统需要提前留接口或边界。

当前文档：

- `10_Public_Deployment_And_Donation_Ops.md`: 公网部署、私域捐助、后台发放权益、云存档与服务端权威经济系统路线。
- `11_Art_Assets_And_Multi_Client_Frontend.md`: 美术资源接入阶段、多端前端形态、资源管理与服务端/前端边界。
