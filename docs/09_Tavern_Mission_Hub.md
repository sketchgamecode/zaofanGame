# 客栈/密谋据点 (Tavern) 核心系统策划案 (v1.0)
**设计宗旨:** 完美还原 S&F 的酒馆大厅交互逻辑，集成日常任务（耗费体力）、离线打工（耗费时间）与资源博弈（赌博）三大核心功能。

---

## 1. 场景一：客栈大厅视窗 (The Tavern View)
当玩家没有正在执行的任务时，进入该模块显示客栈大厅。大厅是一张完整的 2D 背景图，包含 4 个可点击的交互热区（NPC）。

### 1.1 掌柜 (The Bartender)
* **位置:** 通常在吧台后方。
* **功能:** 玩家每天有基础的 100 点“精力/酒量”（对应 S&F 的 Thirst for Adventure）。点击掌柜，可以花费 **1 个【通宝】** 购买一杯“烈酒”（补充 20 点精力）。
* **限制:** 每天最多只能购买 10 次烈酒。UI 需有明显的进度条显示当前剩余精力（如：`80/100`）。

### 1.2 座位上的江湖客 (Quest Givers)
* **位置:** 散坐在客栈的几张桌子旁。每次进入客栈，随机分配 1 个 NPC 高亮可点击。
* **交互:** 点击该 NPC，弹出一个居中的**【三选一任务面板】**。
* **任务生成逻辑:**
  * 每次生成 3 个不同时长的任务（如 2分钟、5分钟、10分钟）。
  * 任务奖励（经验 XP、铜钱 Gold、物品道具）与任务时长、玩家当前等级严格正相关。
  * 面板上需清晰展示：任务文案、所需时间、消耗的精力值、预计收益预览。
  * 玩家选择其中一个并点击“接受”后，直接切换至**场景二：赶路视窗**。

### 1.3 镖局趟子手 / 更夫 (City Guard)
* **位置:** 客栈门口或角落的守卫角色。
* **功能:** 离线挂机打工赚钱。不需要消耗精力，只需要消耗现实时间。
* **交互:** 点击弹出面板，玩家可通过滑动条选择打工时长（1 至 10 小时）。确认后，角色进入打工状态，期间无法接取常规任务。倒计时结束后获得大量铜钱。

### 1.4 千王 / 赌徒 (The Gambler)
* **位置:** 阴暗角落里玩骰子或三仙归洞的角色。
* **功能:** 消耗铜钱或通宝进行双倍押注的迷你游戏（MVP 阶段可暂缓开发，仅留 UI 占位）。

## 2. 场景二：赶路与执行视窗 (The Travel & Quest View)
当玩家在“三选一面板”接受任务后，客栈大厅 UI 立即被隐藏，替换为赶路画面。

### 2.1 视觉表现 (Visuals)
* **背景:** 一张横向卷轴的野外/地牢风景图。
* **角色:** 玩家的虚拟形象（或一个通用的剪影）在画面中央播放简单的原位“行走/奔跑” 序列帧动画。

### 2.2 核心 UI 组件 (UI Elements)
* **倒计时进度条 (Countdown Bar):** 位于画面正下方，巨大的进度条，实时显示剩余时间（如 `04:59`）。
* **【取消】按钮 (Cancel):** 位于左下角。点击后，放弃当前任务，无任何收益，**但不扣除精力**，画面切回客栈大厅。
* **【跳过】按钮 (Skip):** 位于右下角（带有一个沙漏或通宝的 Icon）。点击后，消耗 **1 个【沙漏】道具**或 **1 个【通宝】**，瞬间将倒计时清零。

### 2.3 任务结算 (Quest Resolution)
* 倒计时归零时，触发一场极短的自动战斗（调用 `mathCore.ts` 与 `Arena` 类似的战斗剧本回放）。
* 战斗播放完毕（或玩家点击跳过），弹出奖励结算 Modal（获得经验、铜钱，有概率掉落装备或地下城钥匙）。
* 点击“确认”后，画面切回客栈大厅，等待玩家接取下一个任务。

## 3. 美术资源需求清单 (Art Asset List)
为了实现上述功能，我们需要美术外包（或通过 AI 绘画工具）准备以下 2D 资产。

### 3.1 场景与背景 (Backgrounds)
1. `bg_tavern.png`: 客栈大厅全景图（需预留掌柜、桌椅、角落的空间）。
2. `bg_travel_forest.png` / `bg_travel_night.png`: 至少 2 张赶路用的野外横轴背景图（比例需适配赶路界面的裁切）。

### 3.2 人物与物件 (Sprites & Objects)
1. `npc_bartender.png`: 掌柜立绘（静态，带有一点高亮描边表示可点击）。
2. `npc_quest_givers.png`: 3-5 种不同江湖客的坐姿立绘（和尚、女侠、草寇等）。
3. `npc_guard.png`: 守卫/更夫立绘。
4. `npc_gambler.png`: 赌徒立绘。
5. `anim_player_run.gif` (或序列帧): 玩家角色在赶路界面的通用奔跑动画（MVP 阶段可用单张图带轻微的 CSS 上下浮动代替）。

### 3.3 UI 切图 (UI Elements)
1. `icon_beer.png`: 烈酒/精力图标。
2. `icon_hourglass.png`: 沙漏/加速跳过图标。
3. 任务三选一面板底图 (卷轴或木板质感)。
4. 进度条的底框与填充条贴图 (木质边框，内部为红色或绿色填充)。
