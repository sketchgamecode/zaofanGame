## 1. 核心高级资源定义

为了复刻 Shakes & Fidget (S&F) 的核心商业化/节奏控制机制，我们引入以下两种高级资源：

### 1.1 通宝宝口 (Tokens / Mushrooms)
- **对应 S&F:** 蘑菇 (Mushrooms)
- **定位:** 游戏内最稀缺的硬通货，用于突破时间限制或获取极品资源。
- **获取途径:**
    - 每日任务极低概率掉落 (10%)。
    - 捐助奖励（通过作者下发 Token）。
    - 幸运转盘奖励。

### 1.2 须弥沙漏 (Hourglasses)
- **对应 S&F:** 快速推进沙漏 (Hourglasses / Quick-Sand)
- **定位:** 专门用于跳过任务计时的消耗品。
- **获取途径:**
    - 任务奖励。
    - 黑市购买。
    - 每日签到。

## 2. 消耗机制对齐 (S&F 机制造型)

### 2.1 任务加速 (Quest Acceleration)
- **立即完成**: 消耗 1 个 **通宝**。不论剩余时间多长，直接点击【瞬间完成】。
- **沙漏加速**: 消耗 1 个 **沙漏**。减少 10 分钟任务时长（若任务不足 10 分钟，直接完成且不返环）。

### 2.2 竞技场 (Arena)
- **跳过冷却**: 竞技场战斗有 10 分钟冷却。消耗 1 个 **通宝** 可立即重置冷却进行下一次战斗。
- **次数购买**: 当 3 次免费机会耗尽，可消耗 1 个 **通宝** 购买额外一次挑战机会。

### 2.3 黑市/商店 (Black Market)
- **强制刷新**: 手动点击刷新按钮，固定消耗 1 个 **通宝**。

### 2.4 坐骑/马厩 (Stables) — 待开发
- **顶级坐骑 (赤兔级)**: 租期 14 天，消耗 25 个 **通宝**，提升 50% 任务速度（节省 50% 时间）。

## 3. Data Schema 变更

`GameState` 需要增加以下字段以支持以上逻辑：

```typescript
export interface PlayerResources {
  copper: number;    // 铜钱
  prestige: number;  // 声望
  rations: number;   // 干粮
  tokens: number;    // 通宝 (Mushroom)
  hourglasses: number; // 沙漏
}
```

---

## 4. 后续扩展规划 (Early Game Content)

为了让首日玩家更有沉浸感，我们将按顺序补齐以下 S&F 经典模块：

1. **马厩 (Stables)**: 允许玩家用铜钱或通宝租借坐骑，全局缩短任务耗时。
2. **幸运转盘 (Wheel of Fortune)**: 每日免费抽奖一次，产出经验、金币、通宝、沙漏。
3. **宠物系统 (Pets)**: 50 级解锁，但在早期可以埋下相关的探索任务（掉落蛋）。

## 5. Agent 指导规范
- **禁止**在调试按钮以外的地方提供"免费跳过"功能。
- 所有跳过逻辑必须先检查 `resources.tokens` 或 `resources.hourglasses` 是否充足。
- 消耗通宝时，UI 必须弹出简短的"消耗 1 通宝宝口"浮动文字提示。
