# 核心玩法与系统设计 (v0.1)

## 1. 世界观与核心循环
* **题材:** 玩家扮演一名对大宋朝廷心生不满的江湖人士，从客栈接头暗杀开始，逐步积攒实力，图谋造反。
* **核心循环:** 消耗时间与干粮(接任务) -> 等待倒计时 -> 获取资源(铜钱/声望/装备) -> 消耗铜钱升级属性 / 穿戴更强装备 -> 挑战更难的任务。

## 2. 基础资源与属性定义
### 2.1 基础货币与资源
* **铜钱 (Coins):** 用于升级基础属性、未来在黑市购买装备。
* **声望 (Experience):** 累积满后提升玩家等级。等级影响刷出的任务质量和掉落装备的数值区间。
* **干粮 (Action Points):** 每日上限 100 点。每次执行任务需消耗对应点数。现实时间每日凌晨刷新（MVP 版本可做成手动点击按钮回满，方便测试）。

### 2.2 四大主属性
玩家初始各项属性均为 10，可消耗铜钱无限提升，消耗量随属性等级指数递增。
1. **武力 (STR):** 决定物理攻击力。（文案：力拔山兮气盖世）
2. **智谋 (INT):** 决定法术/暗器攻击力。（文案：运筹帷幄之中）
3. **身法 (AGI):** 决定闪避率与先手速度。（文案：飞檐走壁如履平地）
4. **体质 (END):** 决定最大生命值。（文案：铜皮铁骨百战不殆）

## 3. 核心系统说明

### 3.1 【密谋据点】 (酒馆任务系统)
游戏的核心产出地。
* **刷新机制:** 每次进入面板，接头人（NPC）提供 3 个随机任务。
* **任务模型:**
  * *类型 A (险中求胜):* 高经验，低铜钱，高时间消耗，概率掉装备。
  * *类型 B (劫富济贫):* 高铜钱，低经验，中等时间消耗。
  * *类型 C (跑腿送信):* 平衡收益，极短时间消耗。
* **执行流程:** 点击接受 -> 扣除【干粮】 -> 进入倒计时状态（不可做其他操作） -> 倒计时结束，弹出结算 UI。

### 3.2 【人物行囊】 (装备系统)
极度简化的装备驱动系统。
* **部位设定:** 1. 武器 (增加武力/智谋)
  2. 防具 (增加体质)
  3. 饰品 (增加身法)
  4. 坐骑 (特殊：缩短任务倒计时时间，如“劣马”缩短 5%，“千里马”缩短 20%)
* **装备品质:** 白 (基础数值) -> 绿 (+10% 额外属性) -> 蓝 (+20% 额外属性)。MVP 阶段只投放白绿装。

### 3.3 占位系统 (Placeholder)
以下系统在左侧导航栏可见，但点击后仅显示占位提示，为玩家建立长线期待。
* **【演武场】:** “江湖未定，尔等武艺低微，待等级达到 10 级方可与天下群雄一较高下。” (暗示 PVP 系统)
* **【攻打州府】:** “州府兵力强盛，需集结重兵！此乃后期核心 PVE 玩法，敬请期待下一期更新。” (暗示爬塔/地下城系统)
* **【聚义厅】:** “招兵买马系统筹备中，未来可与兄弟共建山寨。” (暗示公会系统)
* **【黑市】:** “神秘商人进货中...” (暗示商店系统)
